# Controls

## Navigation
- `j` / `Down`: move down
- `k` / `Up`: move up

## Playback
- `Enter`: play selected track
- `p`: play/pause (same as MPRIS PlayPause)
- `n`: next
- `b`: previous
- `q`: quit

## Filtering (search)
- `/`: enter filter mode
- While in filter mode:
  - type to filter track titles (substring match, case-insensitive)
  - `Backspace`: delete last character
  - `Esc`: clear filter and exit filter mode
  - `Enter`: play selected match and exit filter mode
  - `j/k` (or arrows): navigate within filtered results

Notes:
- Filtering changes which tracks are shown in the list.
- Selection always moves inside the currently filtered results.
